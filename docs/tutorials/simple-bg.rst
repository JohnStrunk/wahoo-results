==================================
Creating a simple background image
==================================

.. toctree::
  :hidden:

.. sidebar:: Contents

  .. contents:: Simple background image
    :local:

This tutorial walks through creating a simple background image to enhance the
look of the scoreboard. It is recommended that you follow the earlier tutorial
on :doc:`basic-theme` first.

- Install gimp
- Download the swimmer image
- Create a new image of the correct size
- Set its background to black
- Add the swimmer image as a layer
- Set the color of the swimmer to match the theme
- Save the image as a PNG file
- Import the image into the scoreboard
- Adjust the brightness
